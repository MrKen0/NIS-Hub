# UX Principles

## Mobile-first
- full screen usage on 360px width
- thumb zone primary actions
- minimal horizontal scrolling

## Clarity
- clear labels, high contrast, consistent spacing
- one major action per page (post, RSVP, report)

## Speed
- optimized queries and prefetching
- skeleton loading states
- PWA offline read caches

## Safety and trust
- user status badges (verified, moderator)
- easy report/feedback flows
- transparent content policy and moderation status

## Accessibility
- focus states, semantic HTML, screen reader text
- enough color contrast (4.5:1 for text)
- accessible components in shared library
