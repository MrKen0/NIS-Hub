# Launch Checklist

## Setup
- [ ] Firebase project created + Auth/Firestore/Storage enabled
- [ ] PWA manifest and service worker configured
- [ ] initial data seeding scripts ready

## Quality
- [ ] lint passing (`npm run lint`)
- [ ] build passing (`npm run build`)
- [ ] unit tests passing
- [ ] e2e tests passing

## Security
- [ ] Firestore security rules reviewed and deployed
- [ ] Storage rules reviewed
- [ ] admin role safeguards implemented

## Product
- [ ] MVP flows verified by checklist (auth, post, review)
- [ ] ranking policy and moderation policy documented
- [ ] GDPR-friendly privacy and cookie notices

## Go-live
- [ ] staging smoke test complete
- [ ] monitoring/analytics enabled
- [ ] bug triage plan active
